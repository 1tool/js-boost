# js-boost v1.4.0

Released: 2026-05-21

## Highlights

- Added automatic skill registration during generation.
- Claude skills are now copied from `.ai/skills/` to `.claude/skills/`.
- Non-Claude agent skills are now copied from `.ai/skills/` to `.agents/skills/`.
- Generated agent guidance now references registered skill paths instead of `.ai/skills/`.
- Added Codex MCP registration in `.codex/config.toml`.
- Existing Codex MCP server entries are updated in place instead of duplicated.
- Codex no longer relies on `.mcp.json`; `.mcp.json` is now generated for Claude Code only.
- Watch mode now reacts to changes in `.ai/mcp/mcp.json`.

## Notes

- This release aligns generated Codex MCP configuration with the current Codex documentation for `.codex/config.toml`.
- The CLI version output now matches the package version.
